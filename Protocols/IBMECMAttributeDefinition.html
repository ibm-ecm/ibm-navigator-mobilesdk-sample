IBMECMAttributeDefinition

public protocol IBMECMAttributeDefinition : IBMECMBaseObject

Represents the definition of an attribute, or property, for a class. An IBMECMAttributeDefinition is stored as a collection inside of an IBMECMContentClass.

The IBMECMAttribueDefinition interface provides a template for the metadata for a single property or attribute of a class instance. For example, for propertyA of an item, the metadata describes the property’s type, whether it is required, whether it is hidden, the default value, and other aspects of the property. other metadata.

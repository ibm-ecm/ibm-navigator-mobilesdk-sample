IBMECMDesktop

public protocol IBMECMDesktop : IBMECMBaseObject

Represents an IBM Content Navigator desktop that the current application is accessing. It is usually the second level object after the IBMECMApplication object.

An application can have many desktops and only one default desktop. Typically an application accesses one desktop at a time.

A desktop object manages lists of favorites, repositories, synced items, and other items. It also contains information about server connections and user information. For each desktop, it will also have one default repository.

A desktop must be loaded before it can be used by calling IBMECMApplication.loadDesktop(IBMECMCompletionHandler)

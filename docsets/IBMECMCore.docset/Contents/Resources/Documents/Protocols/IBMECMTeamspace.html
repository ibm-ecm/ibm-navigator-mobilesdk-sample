IBMECMTeamspace

public protocol IBMECMTeamspace : IBMECMBaseObject

Represents a teamspace in IBM Content Navigator.

A teamspace is a restricted view of a repository that allows administrators to control what end users can view and do. It is tied with a teamspace template, which is a starting base for a teamspace instance.

This class represents a teamspace instance and a teamspace template.
